# 🚀 AI Resume Analyzer

A modern React dashboard for resume analysis with authentication and analytics features.

## ✨ Features

- User Login & Registration
- Protected Routes
- Responsive Dashboard
- Sidebar Navigation
- Statistics Cards
- Modern UI with Tailwind CSS

## 🛠️ Tech Stack

- React.js
- React Router DOM
- Tailwind CSS
- Vite
- JavaScript (ES6+)

## ⚙️ Run Project

```bash
npm install
npm run dev
```
